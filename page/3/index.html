スクリーンショットギャラリー

投稿日 2023-03-12 編集日 2024-12-21 # システム

概要

ゲーム内でスクリーンショットを撮影し、記録する機能を提供します。
スクリーンショットギャラリーシーンで、スクリーンショットを閲覧できます。

ゲーム画面のキャプチャをツイートする(2.0.0以降)との連携で、ギャラリーシーンから画像をツイートできます。

※画像へのカーソル画像表示は本プラグインの機能には含まれません。

ゲーム画面のキャプチャをツイートする

投稿日 2023-03-05 編集日 2024-12-21 # システム

概要

ゲーム画面のキャプチャをツイートする機能を提供します。
imgurにゲーム画面のキャプチャをアップロードし、そのURLをツイートする画面をブラウザで表示します。

使用例


利用の前提

本プラグインの利用のためには、imgurのクライアントIDが必要となります。

  1. imgurにアカウントを作成する
  2. Register an Applicationからアプリケーションを作成する


ゲームごとにアプリケーションを作成し、そのクライアントIDをプラグインパラメータに設定してください。

本プラグインはブラウザプレイに対応しません。
