# 概要

2つのプラグインコマンドを提供します。
- パーティメンバーの現在の装備セットを記録する
- パーティメンバーに記録した装備セットを復元する

[GitHub(MV版)](https://github.com/elleonard/DarkPlasma-MV-Plugins/blob/release/DarkPlasma_SaveEquipSet.js)
