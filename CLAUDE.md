日本語で作成された、RPGツクールMZ向けのプラグイン開発用リポジトリです。

# プロジェクトの構造

[README](README.md)を参照してください。

# 設計方針

RPGツクールMZのコアスクリプトをプラグインによって拡張しますが、コアスクリプトで守られているルールは基本的に破りません。
- _で始まるメンバに対してはprivateアクセス以外行いません
- WindowやSpriteなどの表示要素はSceneに依存しません

汎用的な、一般公開するプラグインは src/codes 以下に作成します。
プロジェクト専用の、一般公開しないプラグインは src/excludes 以下に作成します。

src/codes, src/excludes 以下のディレクトリは、それぞれが独立した開発プロジェクトであり、本リポジトリはモノレポ構造を採用しています。

# 開発コマンド

- `yarn generate (プラグイン識別名)` - src/codes以下にプラグインのプロジェクトを作成します
- `yarn generate (プラグイン識別名) e` - src/excludes以下にプラグインのプロジェクトを作成します
- `yarn build:config --target (プラグイン識別名)` - 対象プラグインのconfigをビルドします。 `--exclude` オプションを追加するとsrc/excludes以下のプラグインを対象にします。
- `yarn build --target (プラグイン識別名)` - 対象プラグインをビルドします。 `--exclude` オプションを追加するとsrc/excludes以下のプラグインを対象にします。
